# 📈 Data Analytics

A collection of solo data analytics and machine learning projects built for practice and hands-on learning. Each project takes a real-world dataset through cleaning, exploratory analysis, and — where relevant — predictive modeling and interactive dashboarding.

---

## 📌 Overview

This repository brings together end-to-end analytics projects across churn prediction, retail sales, and public transport intelligence. The work is primarily done in **Python / Jupyter Notebook**, combining data analysis, visualization, and machine learning.

---

## 🗂️ Projects

### 🔮 Customer Churn Prediction Dashboard
Predicts which customers are likely to churn using machine learning, paired with an interactive dashboard to explore risk drivers and segment-level churn patterns.

### 🛍️ Retail Store Sales And Customer Behaviour Analysis
Analyzes retail transaction data to understand sales performance and customer behavior — spending patterns, product trends, and segment insights that inform business decisions.

### 🚌 Smart Public Transport Analytics And Business Intelligence Dashboard
A business-intelligence project on public transport data, surfacing ridership trends, route/operational insights, and BI-style visualizations to support smarter transport planning.

## 🛠️ Tech Stack

- **Python** — core language for analysis and modeling
- **Jupyter Notebook** — exploratory analysis and workflow
- **pandas / NumPy** — data cleaning and manipulation
- **matplotlib / seaborn** — visualization
- **scikit-learn** — machine learning (e.g., churn prediction)
- **Streamlit / dashboard tooling** — interactive dashboards (see `runtime.txt`)

---

## 🚀 How to Use

1. Open any project folder above.
2. Launch the `.ipynb` notebook in Jupyter to follow the analysis end to end.
3. For dashboard projects, run the app locally (e.g., `streamlit run app.py`) or deploy using the included `runtime.txt`.

---

## 🎯 Purpose

These projects demonstrate skills in data cleaning, exploratory data analysis, machine learning, and turning data into interactive, business-ready insights.
